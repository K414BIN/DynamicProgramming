1. Задача
Для прямоугольного поля размера M на N клеток, подсчитать количество путей из верхней левой клетки в правую нижнюю. Известно что ходить можно только на одну клетку вправо или вниз. 
Решил через формулу комбинаторики ( см. formula.png ) - формула неупорядоченной выборки с повторениями.
Хотя на уроке я писал в чате, что количество ходов по полю M x N это соответствующая ячейка в треугольнике Паскаля, все же решил найти более научное решение.
Зато сделал вычисление факториала через метод бинарного дерева.
Использовал лекции "Комбинаторные алгоритмы для программистов."  Костюкова Нина Ивановна  Курсы ИНТУИТ.
